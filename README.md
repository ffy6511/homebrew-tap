# homebrew-tap

This repository hosts the Homebrew formulas for my command-line tools.

## Included formulae

- `aht-cli`: CLI for Adjust Heading in Tree. It lets you inspect and edit Markdown or Typst heading trees from the terminal.

## Install

```bash
brew tap ffy6511/tap
brew install aht-cli
```

You can also use the fully qualified formula name:

```bash
brew install ffy6511/tap/aht-cli
```
